# RAG機能テストドキュメント

このファイルはLlamaAppのRAGファイル対応機能をテストするためのサンプルドキュメントです。

## 機能概要

LlamaAppは**ローカルLLMチャットアプリケーション**です。主な特徴:

- node-llama-cppによるローカル推論
- Metal GPU加速対応
- プライバシー重視の完全ローカル実行

## RAG機能の拡張

Phase 1では以下の機能が追加されました:

### 対応ファイル形式
- **TXT**: プレーンテキストファイル
- **MD**: マークダウンファイル

### 実装コンポーネント
1. `FileFetcher` - ローカルファイル読み込み
2. `RagManager` - ファイル処理統合
3. IPC通信 - セキュアなファイルアクセス
4. UI - ファイル選択ボタン
